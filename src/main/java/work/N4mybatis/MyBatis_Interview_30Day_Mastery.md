# MyBatis 30天面试突击指南 (🗺️ ORM 映射与源码深度进阶版)

> **总纲**：本指南深度聚焦于 **MyBatis**。我们将从其作为半自动 ORM 框架的核心设计出发，全面攻克动态 SQL 解析、参数映射机制、一级/二级缓存原理、插件扩展（Interceptor）、延迟加载以及最核心的 SqlSession 运行原理与 Mapper 接口代理机制。
> 
> **学习路径**：
> 1. **Week 1 (1-7天)**：核心配置、动态 SQL 与参数处理。
> 2. **Week 2 (8-14天)**：结果映射（ResultMap）、级联查询与延迟加载。
> 3. **Week 3 (15-21天)**：缓存机制（一级、二级缓存）与缓存失效深度分析。
> 4. **Week 4 (22-26天)**：插件机制（拦截器）原理、PageHelper 分页原理。
> 5. **Week 5 (27-30天)**：源码解析：SqlSession 运行全生命周期、Mapper 代理实现与整合 Spring 原理。
