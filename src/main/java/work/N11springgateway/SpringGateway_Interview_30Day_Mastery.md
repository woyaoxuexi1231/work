# Spring Cloud Gateway 30天面试突击指南 (🌉 微服务流量守卫专家版)

> **总纲**：本指南深度聚焦于 **Spring Cloud Gateway**。我们将从其基于 Spring 5, Project Reactor 和 Spring Boot 2.0 的响应式架构出发，全面攻克断言（Predicate）、过滤器（Filter）、动态路由、限流熔断、整合安全以及生产级监控调优。
> 
> **学习路径**：
> 1. **Week 1 (1-7天)**：响应式编程基础、核心架构与路由配置。
> 2. **Week 2 (8-14天)**：断言工厂与过滤器工厂深度应用。
> 3. **Week 3 (15-21天)**：动态路由、服务发现集成与跨域处理。
> 4. **Week 4 (22-26天)**：高阶特性：限流（Redis/Sentinel）、熔断降级与整合安全。
> 5. **Week 5 (27-30天)**：性能调优、监控指标（Actuator）与大厂网关实战方案。

---
