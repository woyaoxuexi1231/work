# RocketMQ 30天面试突击指南 (🚀 金融级可靠消息专家版)

> **总纲**：本指南深度聚焦于 **Apache RocketMQ**。我们将从其独特的架构设计出发，全面攻克消息存储模型、各种高级消息类型（事务、顺序、延时）、高可用机制、高性能吞吐原理以及生产级故障排查。
> 
> **学习路径**：
> 1. **Week 1 (1-7天)**：核心架构、NameServer 设计与基础收发流程。
> 2. **Week 2 (8-14天)**：消息类型全解：顺序、延时、批量与事务消息。
> 3. **Week 3 (15-21天)**：存储模型深度剖析：CommitLog, ConsumeQueue 与 mmap 零拷贝。
> 4. **Week 4 (22-26天)**：高可用与集群：负载均衡、Rebalance、主从同步与 Dledger。
> 5. **Week 5 (27-30天)**：性能调优、监控管理与大厂消息架构实战。
