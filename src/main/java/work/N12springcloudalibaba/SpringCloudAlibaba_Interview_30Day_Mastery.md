# Spring Cloud Alibaba 30天面试突击指南 (🏮 国产微服务巅峰版)

> **总纲**：本指南深度聚焦于 **Spring Cloud Alibaba (SCA)** 体系。我们将从阿里巴巴双11高并发实战经验出发，全面攻克 Nacos 服务治理、Sentinel 流量防护、Seata 分布式事务、RocketMQ 消息驱动以及 Dubbo 生态集成。
> 
> **注意**：本指南专门针对阿里系开源组件，强调高性能、高可用以及应对极端流量峰值的能力。
> 
> **学习路径**：
> 1. **Week 1 (1-7天)**：Nacos 深度实践：从注册中心到配置中心 (Raft/Distro)。
> 2. **Week 2 (8-14天)**：Sentinel 流量治理：限流、熔断、降级与热点防护。
> 3. **Week 3 (15-21天)**：Seata 分布式事务：AT、TCC、Saga 与 XA 模式全解。
> 4. **Week 4 (22-26天)**：RocketMQ 消息中间件：高吞吐、可靠性与事务消息。
> 5. **Week 5 (27-30天)**：Dubbo 集成、Sidecar 异构集成与双11级别架构调优。

---
