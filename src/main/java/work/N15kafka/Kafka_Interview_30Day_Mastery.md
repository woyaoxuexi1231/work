# Kafka 30天面试突击指南 (🎸 高吞吐流处理专家版)

> **总纲**：本指南深度聚焦于 **Apache Kafka**。我们将从其颠覆性的高吞吐设计出发，全面攻克分布式分区模型、ISR 副本机制、零拷贝技术、控制器（Controller）选举、流处理（Kafka Streams）以及海量数据下的性能调优。
> 
> **学习路径**：
> 1. **Week 1 (1-7天)**：核心架构、分区模型与消息生产全链路。
> 2. **Week 2 (8-14天)**：消费者组原理、Rebalance 与位移管理。
> 3. **Week 3 (15-21天)**：副本机制、ISR 协议、高水位（HW）与 LEO 深度解析。
> 4. **Week 4 (22-26天)**：存储内核、索引机制、零拷贝与日志清理策略。
> 5. **Week 5 (27-30天)**：高可用集群管理、监控调优与 Kafka 生态（Connect/Streams）。
