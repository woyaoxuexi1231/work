知识总结

一图掌握 JVM 运行时数据区域核心要点

1 线程私有 vs 线程共享

PRIVATE

程序计数器、虚拟机栈、本地方法栈与线程生命周期绑定。每个线程独立拥有，数据互不干扰，无需考虑线程安全问题。

SHARED

堆、方法区被所有线程共享。堆存放对象实例，需 GC 回收；方法区存储类元数据，JDK 8 后元空间使用本地内存。

2 异常记忆口诀

🧠 "栈可溢，堆可满，计数器最稳"
  • • 虚拟机栈 & 本地方法栈：StackOverflowError（深度）+ OutOfMemoryError（扩展）
  • • 堆 & 方法区：只有 OutOfMemoryError
  • • 程序计数器：无任何异常
⚡ JDK 8 关键变化

永久代（PermGen）被移除，方法区实现改为元空间（Metaspace），使用本地内存而非 JVM 堆内存，默认无上限（受限于系统内存）。
