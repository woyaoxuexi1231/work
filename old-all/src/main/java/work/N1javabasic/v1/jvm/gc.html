JVM 垃圾回收分代假说 · 对象生命周期

✓ Eden: 新对象分配 (Age=0)  |  ✓ Minor GC: Eden + 活跃Survivor → 目标Survivor (Age+1)  |  ✓ 晋升: Age ≥ 3 → 老年代  |  ✓ Full GC: 全堆回收

Eden 区 · 诞生之地
所有新对象在 Eden 分配，年龄初始为 0。Eden 满时触发 Minor GC
Survivor 区 · 复制轮回
Minor GC 时存活对象在 From/To 之间复制，每次复制年龄 +1。同一时刻仅一个Survivor活跃
老年代 · 长寿归宿
年龄 ≥ 3 或 Survivor 溢出时晋升到老年代。Full GC 时全堆回收
