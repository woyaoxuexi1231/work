锁升级触发条件 & 对象头标志位速查手册

🔁 升级触发逻辑
  • 1 无锁 → 偏向锁 : 默认开启偏向锁（JDK6+但有延迟），当对象第一个线程获取锁时置为偏向模式。
  • 2 偏向锁 → 轻量级锁 : 其他线程竞争时，偏向锁撤销升级为轻量级锁（CAS尝试获取）。
  • 3 轻量级锁 → 重量级锁 : 自旋超过阈值（自适应自旋）或竞争线程数过多，膨胀为重量级锁，线程阻塞。
