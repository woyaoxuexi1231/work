Part 1 · MultipartResolver 基本用法

// Controller
@PostMapping("/upload")
public String upload(@RequestParam("file") MultipartFile file) {
    file.transferTo(new File("/uploads/" + file.getOriginalFilename()));
    return "ok";
}

// 配置
spring.servlet.multipart.max-file-size=10MB
spring.servlet.multipart.max-request-size=20MB

Spring Boot 默认用 StandardServletMultipartResolver（Servlet 3.0+ 原生支持）。旧项目用 CommonsMultipartResolver（需 commons-fileupload 依赖）。

Part 2 · 面试追问

Q

🔧 大文件上传 OOM 怎么办？

不要用 transferTo 全量加载到内存。设置 spring.servlet.multipart.file-size-threshold=1MB，小于阈值存内存，大于阈值写临时文件。超大文件用分片上传+断点续传，不要走 MultipartFile。
