📅 HTTP 缓存控制与 ETag

CacheControl、ETag、Last-Modified — Spring MVC 的 HTTP 缓存策略

Part 1 · 两种缓存机制

// CacheControl（强制缓存）
return ResponseEntity.ok()
    .cacheControl(CacheControl.maxAge(1, TimeUnit.HOURS))
    .body(user);

// ETag（条件缓存）— ShallowEtagHeaderFilter
@Bean ShallowEtagHeaderFilter etagFilter() { return new ShallowEtagHeaderFilter(); }
// 响应体 MD5 → ETag header → 客户端下次带 If-None-Match → 没变返回 304

Part 2 · 面试追问

Q

ETag 和 Last-Modified 什么区别？

ETag 是内容的指纹（MD5），内容变了 ETag 变。Last-Modified 是时间戳，精度到秒。ETag 更精确，但计算成本更高。ShallowEtagHeaderFilter 会对整个响应体算 MD5——大响应体慎用。
