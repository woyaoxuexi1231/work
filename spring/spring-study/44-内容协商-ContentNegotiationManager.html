Part 2 · 触及灵魂

💡 为什么要禁用路径扩展名？

Spring Boot 2.6+ 默认禁用 suffix 匹配。原因是安全问题：/users.json 和 /users 走同一个 Controller，但 Accept header 不同——可能导致 RFD 攻击。同时 Restful API 中 URL 应该只表示资源，格式由 Header 决定。

Part 3 · 面试追问

Q

默认用 JSON，怎么改成默认 XML？

ContentNegotiationConfigurer.defaultContentType(MediaType.APPLICATION_XML)。但不推荐——JSON 是今天的事实标准。
