以下为 **Spring Framework** 与 **Spring Boot** 核心知识点的全面列表，仅聚焦两者自身，不涉及 Spring Security、Spring Cloud、Spring Data 等衍生项目。

---

### 一、Spring Framework 知识点

**1. 核心容器 (Core Container)**
- IoC 容器：`BeanFactory`、`ApplicationContext`
- 依赖注入：构造器注入、Setter 注入、字段注入
- Bean 定义与元数据：XML、注解、Java 配置类
- Bean 作用域：`singleton`、`prototype`、`request`、`session`、`application`、`websocket`
- Bean 生命周期：实例化、属性填充、`BeanPostProcessor`、初始化回调、销毁回调
- 延迟加载：`@Lazy`
- 自动装配：`@Autowired`、`@Qualifier`、`@Primary`、`@Resource`
- 组件扫描：`@ComponentScan`、`@Component`、`@Service`、`@Repository`、`@Controller`
- Java 配置：`@Configuration`、`@Bean`、`@Import`
- 属性注入：`@Value`、`@PropertySource`
- SpEL (Spring 表达式语言)
- 资源抽象：`Resource`、`ResourceLoader`
- 环境抽象：`Environment`、`PropertySource`、`Profile`
- 国际化：`MessageSource`
- 事件机制：`ApplicationEvent`、`ApplicationListener`、`@EventListener`、`ApplicationEventPublisher`
- 类型转换：`ConversionService`、`Converter`、`Formatter`
- 数据校验：`Validator`、`Errors`、Bean Validation 集成

**2. AOP (面向切面编程)**
- AOP 核心概念：切面、连接点、通知、切入点、引入
- 通知类型：`@Before`、`@AfterReturning`、`@AfterThrowing`、`@After`、`@Around`
- 切入点表达式：`execution`、`within`、`this`、`target`、`args`、`@annotation`
- 引入：`@DeclareParents`
- AspectJ 集成与 `@Aspect`
- 代理机制：JDK 动态代理、CGLIB 代理
- `ProxyFactoryBean`、`DefaultAopProxyFactory`
- AOP 自动代理：`@EnableAspectJAutoProxy`

**3. 数据访问与事务管理**
- 数据访问异常体系：`DataAccessException` 及其子类
- JDBC 抽象：`JdbcTemplate`、`NamedParameterJdbcTemplate`
- 数据源配置与 `DataSourceUtils`
- ORM 集成：`LocalSessionFactoryBean`、`LocalContainerEntityManagerFactoryBean`、`HibernateJpaVendorAdapter`
- 事务管理抽象：`PlatformTransactionManager`
- 声明式事务：`@Transactional`（传播行为、隔离级别、超时、回滚规则）
- 编程式事务：`TransactionTemplate`
- 事务同步与资源绑定：`TransactionSynchronizationManager`

**4. Spring MVC (Web)**
- 前端控制器：`DispatcherServlet`
- 控制器：`@Controller`、`@RestController`
- 请求映射：`@RequestMapping`、`@GetMapping`、`@PostMapping` 等
- 请求参数绑定：`@RequestParam`、`@PathVariable`、`@RequestHeader`、`@CookieValue`、`@RequestBody`
- 模型与视图：`Model`、`ModelMap`、`ModelAndView`、`@ModelAttribute`
- 视图解析：`ViewResolver`、`InternalResourceViewResolver`、`ContentNegotiatingViewResolver`
- 数据绑定：`@InitBinder`、`WebDataBinder`
- 类型转换：`Converter`、`Formatter`、`FormattingConversionService`
- 数据校验：`@Valid`、`@Validated`、`Validator`
- HTTP 消息转换：`HttpMessageConverter`
- 内容协商：`ContentNegotiationManager`
- 静态资源处理：`ResourceHttpRequestHandler`
- 拦截器：`HandlerInterceptor`
- 异常处理：`@ExceptionHandler`、`@ControllerAdvice`、`HandlerExceptionResolver`
- 跨域支持：`@CrossOrigin`、`CorsFilter`
- 文件上传：`MultipartResolver`
- 异步请求：`Callable`、`DeferredResult`、`AsyncTask`
- WebSocket 支持：`WebSocketHandler`、STOMP、`@MessageMapping`
- HTTP 缓存控制与 ETag
- 路径匹配：`PathPattern`、`AntPathMatcher`

**5. Spring WebFlux (响应式 Web)**
- 响应式核心：`Mono`、`Flux`（Project Reactor 集成）
- 响应式 Web 栈：`WebFlux`、`DispatcherHandler`
- 注解驱动编程：`@RestController` 返回响应式类型
- 函数式编程模型：`RouterFunction`、`HandlerFunction`、`RouterFunctions`
- 响应式 Web 客户端：`WebClient`
- 响应式数据访问：`R2DBC`（连接工厂、事务）
- 响应式适配：`WebFluxConfigurer`

**6. 测试支持 (spring-test)**
- 测试上下文框架：`TestContext`、`TestExecutionListener`
- JUnit 集成：`SpringExtension`（JUnit 5）、`@SpringJUnitConfig`
- 上下文缓存：`@ContextConfiguration`、`@DirtiesContext`
- 测试属性：`@TestPropertySource`
- 事务测试：`@Transactional`、`@Commit`、`@Rollback`
- 模拟请求：`MockMvc`、`MockHttpServletRequest`
- 上下文自定义：`@ActiveProfiles`、`@Sql`、`@SqlGroup`
- 反射测试工具：`ReflectionTestUtils`

**7. 缓存抽象**
- 缓存注解：`@Cacheable`、`@CacheEvict`、`@CachePut`、`@Caching`
- 缓存管理器：`CacheManager`、`ConcurrentMapCacheManager`
- 缓存策略：条件缓存、key 生成器（`KeyGenerator`）
- 缓存同步：`@CacheConfig`

**8. 任务调度与异步执行**
- `TaskExecutor`、`TaskScheduler`
- 注解：`@Scheduled`、`@Async`
- 配置：`@EnableAsync`、`@EnableScheduling`

**9. 消息支持 (JMS)**
- `JmsTemplate`
- 消息驱动 POJO：`@JmsListener`、`@EnableJms`
- 连接工厂与目的地抽象

**10. 其他**
- JMX 支持：`@ManagedResource`、`MBeanExporter`
- 资源属性编辑与注册：`PropertyEditor`、`CustomEditorConfigurer`
- 远程调用：`HttpInvoker`、`RMI`（已不推荐，但仍为 Framework 一部分）
- 仪器与类加载：`Instrumentation`、`LoadTimeWeaver`
- SpEL 高级用法

---

### 二、Spring Boot 知识点

**1. 核心机制**
- 起步依赖：`spring-boot-starter-*`
- 自动配置：`@EnableAutoConfiguration`、`@SpringBootApplication`
- 启动类：`SpringApplication`
- 条件注解：`@Conditional`、`@ConditionalOnClass`、`@ConditionalOnMissingBean`、`@ConditionalOnProperty` 等
- 自动配置报告：`debug=true` 输出条件评估详情

**2. 外部化配置与 Profile**
- 配置文件：`application.properties`、`application.yml`
- 属性加载顺序与源：命令行、环境变量、`application-{profile}.properties`、`@PropertySource` 等
- 类型安全配置：`@ConfigurationProperties`、`@EnableConfigurationProperties`
- 宽松绑定与属性转换
- Profile 与多环境配置：`spring.profiles.active`
- 配置元数据处理器：`spring-boot-configuration-processor`

**3. Web 开发**
- 嵌入式 Servlet 容器：Tomcat、Jetty、Undertow
- Spring MVC 自动配置：`WebMvcAutoConfiguration`
- 静态资源目录映射、欢迎页、Favicon 配置
- 模板引擎自动配置：Thymeleaf、FreeMarker
- 错误处理：`/error` 端点、`ErrorAttributes`、`ErrorController`、自定义错误页面
- 内容协商与消息转换器自动配置
- 跨域全局配置：`WebMvcConfigurer` 中的 `addCorsMappings`
- 路径匹配策略配置：`spring.mvc.pathmatch`
- 嵌入式容器定制：`WebServerFactoryCustomizer`
- WebFlux 自动配置：`WebFluxAutoConfiguration`
- 响应式静态资源与模板

**4. 数据访问相关自动配置**
- 数据源自动配置：`DataSourceAutoConfiguration`
- 连接池：HikariCP 默认连接池，可配置 DBCP2 等
- `JdbcTemplate` 自动配置
- 事务管理器自动配置：`TransactionAutoConfiguration`
- 数据库初始化：`spring.sql.init`、`schema.sql`、`data.sql`
- 数据源指标与健康检测

**5. 监视与管理 (Actuator)**
- Actuator 端点：`/health`、`/info`、`/metrics`、`/env`、`/loggers`、`/mappings`、`/beans` 等
- 自定义端点：`@Endpoint`、`@WebEndpoint`
- 健康指示器：`HealthIndicator`、`AbstractHealthIndicator`
- 应用信息：`InfoContributor`
- 指标与监控：Micrometer 集成、`MeterRegistry`
- 审计：`AuditEventRepository`、`AbstractAuthenticationAuditListener`
- HTTP 跟踪：`HttpTraceRepository`
- 端点安全与暴露控制：`management.endpoints.web.exposure`

**6. 测试支持**
- `@SpringBootTest`：完整集成测试
- 测试切片：`@WebMvcTest`、`@WebFluxTest`、`@JdbcTest`、`@JsonTest`、`@RestClientTest`
- 模拟 Bean：`@MockBean`、`@SpyBean`
- 测试环境：`@TestConfiguration`、`@TestPropertySource`
- 测试工具：`TestRestTemplate`、`MockMvc`
- 输出捕获：`OutputCaptureExtension`

**7. 开发者工具 (DevTools)**
- 自动重启与类路径更改检测
- 条件禁用的静态资源缓存
- LiveReload 支持
- 远程调试与更新：`spring-boot-devtools` 远程模式
- 全局配置文件：`$HOME/.spring-boot-devtools.properties`

**8. 打包与运行**
- 可执行 JAR/WAR：Maven 插件、Gradle 插件
- 启动器脚本支持：完全可执行 JAR
- 构建信息：`META-INF/build-info.properties`
- 多模块打包
- Spring Boot CLI 快速原型

**9. 日志**
- 默认 Logback，支持 log4j2、java.util.logging
- 日志级别分组配置：`logging.level`
- 日志文件输出与归档：`logging.file`、`logging.pattern`
- 按 Profile 配置日志

**10. 其他特性**
- 启动事件与回调：`ApplicationRunner`、`CommandLineRunner`，执行顺序 `@Order`
- 自定义 Banner：`banner.txt`
- 失败分析器：`FailureAnalyzer`、`FailureAnalysis`
- 环境后处理器：`EnvironmentPostProcessor`
- 自定义 Starter 编写：`autoconfigure` 模块，`spring.factories` 注册
- JSON 处理：`JacksonAutoConfiguration`、自定义 `ObjectMapper`
- Quartz 调度器自动配置
- 邮件发送：`JavaMailSender` 自动配置
- 参数验证：自动启用方法级别验证
- 安全：基础认证与安全自动配置仅指 `spring-boot-autoconfigure` 中极简的安全基础，不涉及 Spring Security 的任何细节（仅默认的用户密码生成等）。

---

以上列表覆盖了 Spring Framework 和 Spring Boot 自身功能边界内的所有主要知识点，未引入任何其他 Spring 顶级项目。
