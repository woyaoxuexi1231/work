JDK JDK 动态代理 — Proxy.newProxyInstance

核心 API：Proxy.newProxyInstance(ClassLoader, interfaces[], InvocationHandler)。 在运行时动态生成一个实现了指定接口的代理类，所有方法调用被转发到 InvocationHandler.invoke()。

MyBatis MyBatis-Spring 完整链路 — 从接口到 SQL

MyBatis 使用 FactoryBean + JDK 动态代理。每个 Mapper 接口对应一个 MapperFactoryBean， getObject() 返回 JDK 代理对象。代理内部的 MapperProxy 负责"接口方法 → SQL 语句"的转换。

① Spring 启动 → 扫描 @MapperScan → 找到 UserMapper 接口
② 为 UserMapper 注册 MapperFactoryBean (实现 FactoryBean)
③ Spring getBean("userMapper") → 触发 MapperFactoryBean.getObject()
④ getObject() 内部调用 Proxy.newProxyInstance → 生成 $Proxy
⑤ $Proxy 的 InvocationHandler = MapperProxy
⑥ 调用 userMapper.findById(1L) → MapperProxy.invoke() 拦截
⑦ MapperProxy 根据方法名找到 SQL → 通过 SqlSession 执行 → 返回结果
