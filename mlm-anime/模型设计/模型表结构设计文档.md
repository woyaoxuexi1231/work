# 模型表结构设计文档

## 1. 表结构

### `model` 表

| 字段 | 类型 | 必填 | 说明 |
|------|------|:--:|------|
| `id` | BIGINT | ✓ | 主键，自增 |
| `model_id` | VARCHAR(128) | ✓ | 模型唯一标识，如 `doubao-seed-2-0-pro-260215` |
| `model_name` | VARCHAR(128) | ✓ | 模型显示名称，如 `Doubao Seed 2.0 Pro` |
| `provider` | VARCHAR(32) | ✓ | 提供商：`VOLCENGINE`、`DEEPSEEK` |
| `model_type` | VARCHAR(16) | ✓ | 模型类型：`TEXT`、`IMAGE`、`VIDEO` |
| `status` | VARCHAR(16) | ✓ | 状态：`ACTIVE`（可用）、`DEPRECATED`（已弃用） |
| `is_recommended` | TINYINT(1) | | 是否推荐，0=否，1=是 |
| `description` | VARCHAR(512) | | 模型简介 |
| `detail_url` | VARCHAR(512) | | 模型详情页链接 |
| `base_url` | VARCHAR(512) | | API 地址，火山引擎模型可不填 |
| `sort_order` | INT | | 排序权重，越大越靠前 |
| `created_at` | DATETIME | ✓ | 创建时间 |
| `updated_at` | DATETIME | ✓ | 更新时间 |

## 2. DDL

```sql
CREATE TABLE `model` (
    `id`              BIGINT        NOT NULL AUTO_INCREMENT COMMENT '主键',
    `model_id`        VARCHAR(128)  NOT NULL COMMENT '模型唯一标识',
    `model_name`      VARCHAR(128)  NOT NULL COMMENT '模型显示名称',
    `provider`        VARCHAR(32)   NOT NULL COMMENT '提供商：VOLCENGINE / DEEPSEEK',
    `model_type`      VARCHAR(16)   NOT NULL COMMENT '模型类型：TEXT / IMAGE / VIDEO',
    `status`          VARCHAR(16)   NOT NULL DEFAULT 'ACTIVE' COMMENT '状态：ACTIVE / DEPRECATED',
    `is_recommended`  TINYINT(1)    NOT NULL DEFAULT 0 COMMENT '是否推荐',
    `description`     VARCHAR(512)  NULL COMMENT '模型简介',
    `detail_url`      VARCHAR(512)  NULL COMMENT '模型详情页链接',
    `base_url`        VARCHAR(512)  NULL COMMENT 'API 地址',
    `sort_order`      INT           NOT NULL DEFAULT 0 COMMENT '排序权重',
    `created_at`      DATETIME      NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    `updated_at`      DATETIME      NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    PRIMARY KEY (`id`),
    UNIQUE KEY `uk_model_id_provider` (`model_id`, `provider`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='模型表';
```
