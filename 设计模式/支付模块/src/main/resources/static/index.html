架构要点

策略自动注册

PaymentStrategyRegistry 通过 List<PaymentStrategy> 接收 Spring 注入的所有实现类，构建 Map 做 O(1) 路由。 新渠道只需新建 @Component 实现类即可，零修改已有代码。

责任链异常控制

SkipException → 跳过当前节点继续链 · CallbackException → 终止链 · 其他异常 → 终止链。 链顺序通过 @Order(n) 控制， 增删调序无需修改其他 Handler。
