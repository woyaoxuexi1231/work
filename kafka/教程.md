## Kafka + Spring Boot 完整学习路线图

### 🟢 第一阶段：基础入门（第 1-4 天）
| 天数  | 题目                                     | 核心内容                                                     |
| ----- | ---------------------------------------- | ------------------------------------------------------------ |
| 第1天 | Kafka 与 Spring Boot 初体验              | 安装 Kafka、创建 Spring Boot 项目、引入 spring-kafka、发送第一条消息、消费第一条消息 |
| 第2天 | 核心概念：主题、分区、偏移量             | 主题创建、分区作用、偏移量含义、查看和管理工具（kafka-topics） |
| 第3天 | 生产者详解（配置、发送方式）             | 同步/异步发送、生产者配置（acks、retries、batch.size、linger.ms）、拦截器 |
| 第4天 | 消费者详解（订阅、反序列化、提交偏移量） | 订阅主题/分区、自动/手动提交偏移量、反序列化器、消费线程模型 |

### 🟡 第二阶段：进阶特性（第 5-10 天）
| 天数   | 题目                                  | 核心内容                                                     |
| ------ | ------------------------------------- | ------------------------------------------------------------ |
| 第5天  | 消费者组与再均衡                      | 消费者组概念、分区再均衡过程、再均衡监听器、分配策略（Range/RoundRobin/Cooperative） |
| 第6天  | 精确一次语义（EOS）与幂等生产者       | 幂等性（enable.idempotence）、事务生产者、Exactly-Once 语义、事务 API 在 Spring Boot 中的实现 |
| 第7天  | Spring Kafka 中的消息转换与序列化进阶 | 自定义序列化器/反序列化器（Avro/JSON）、ErrorHandlingDeserializer、信任包配置 |
| 第8天  | 生产者与消费者中的异常处理            | 生产者重试与回调、消费者异常（SeekToCurrentErrorHandler、DeadLetterPublishingRecoverer）、死信队列（DLQ） |
| 第9天  | Kafka 消息头与路由                    | 消息头的使用、基于头的路由消费、使用 @Header 注解            |
| 第10天 | 重平衡与静态成员关系                  | 静态成员关系（group.instance.id）、避免不必要的再均衡、会话超时与心跳参数调优 |

### 🔴 第三阶段：高级实战（第 11-16 天）
| 天数   | 题目                                     | 核心内容                                                     |
| ------ | ---------------------------------------- | ------------------------------------------------------------ |
| 第11天 | 非阻塞重试与 RetryTopicConfiguration     | 使用 @RetryableTopic 注解、配置重试主题、延迟主题（Dlt）、自动创建死信主题 |
| 第12天 | Kafka 事务端到端 Exactly-Once            | 跨生产者-消费者事务、只读事务隔离级别、Kafka 事务在微服务中的使用模式 |
| 第13天 | 分区再均衡与状态存储                     | 使用 StateStore（例如在交互式查询中）、消费者再均衡时保存/恢复偏移量状态 |
| 第14天 | Kafka Streams 入门与 Spring Cloud Stream | 什么是 Kafka Streams、Spring Cloud Stream Kafka Binder、简单流处理（map/filter/join） |
| 第15天 | Kafka Streams 状态存储与窗口操作         | 状态存储（RocksDB）、窗口聚合（滚动窗口、跳跃窗口）、KTables 与全局表 |
| 第16天 | Spring Kafka 与可观测性                  | Micrometer 集成、Kafka 客户端指标（消费者/生产者指标）、自定义健康检查与监控端点 |

### 🔵 第四阶段：生产环境调优与生态（第 17-22 天）
| 天数   | 题目                                        | 核心内容                                                     |
| ------ | ------------------------------------------- | ------------------------------------------------------------ |
| 第17天 | 生产者性能调优                              | 压缩（gzip/snappy/lz4/zstd）、缓冲区调优、批量与延迟权衡、流量控制与背压 |
| 第18天 | 消费者性能调优                              | fetch.min.bytes、fetch.max.wait.ms、max.poll.records、并发消费者（ConcurrentMessageListenerContainer） |
| 第19天 | Kafka 连接器（Kafka Connect）与 Spring Boot | 连接器概念、JDBC Sink Connector、Debezium CDC、使用 Spring Boot 管理连接器 |
| 第20天 | 安全配置：SSL/SASL 与 ACL                   | 配置 SSL 加密、SASL/PLAIN 认证、授权（ACL）、Spring Boot 中安全配置的加载 |
| 第21天 | 多集群、镜像与灾难恢复                      | MirrorMaker 2、多主题消费、跨数据中心架构、Spring Boot 多集群生产/消费者配置 |
| 第22天 | 综合实战项目                                | 设计一个“订单事件驱动系统”：订单创建 -> 库存扣减 -> 支付通知，使用事务消息与 Streams 聚合报表 |
