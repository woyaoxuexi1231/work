架构要点

策略自动注册

PaymentStrategyRegistry 通过 List<PaymentStrategy> 接收 Spring 注入的所有实现类，新渠道只需加 @Component。

每渠道独立责任链

CallbackChannelStrategy 定义该渠道的链步骤。支付宝7步全走，微信6步（无风控），银联5步（无幂等+风控）。 换渠道时链实时切换。

异常三态控制

SkipException → 跳过继续 · CallbackException → 终止链 · 链顺序由 getHandlerNames() 控制。
